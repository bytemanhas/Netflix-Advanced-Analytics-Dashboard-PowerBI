# Netflix-Advanced-Analytics-Dashboard-PowerBI
A comprehensive 6-page Power BI analytics solution featuring Executive Overview, Customer Behavior Analytics, Customer 360° Insights, Content Performance Analysis, Churn &amp; Retention Monitoring, and Global Viewership Analytics using DAX, data modeling, and interactive visualizations.





# 🎬 Netflix Advanced Analytics Dashboard | Power BI

## Project Overview

The Netflix Advanced Analytics Dashboard is a comprehensive Business Intelligence solution developed in Power BI to analyze customer behavior, content performance, retention trends, and global viewership patterns.

The dashboard transforms raw streaming data into actionable business insights through interactive visualizations, KPI monitoring, customer segmentation, churn analysis, and geographic reporting.

This project demonstrates advanced Power BI capabilities including Data Modeling, DAX Calculations, Power Query transformations, KPI development, and dashboard storytelling.

---

#  Project Objectives

* Analyze customer viewing behavior and engagement.
* Track overall platform performance using KPIs.
* Identify high-performing content and genre trends.
* Monitor customer retention and churn risk.
* Develop Customer 360° analytical views.
* Evaluate global viewership and regional preferences.
* Support data-driven decision making through interactive reporting.

# 🗂 Dataset Information

The dataset contains Netflix streaming activity data including:

### Customer Data

* Customer Name
* Country
* Subscription Plan

### Content Data

* Title ID
* Genre
* Content Type

### Viewing Data

* Watch Date
* Watch Time (Minutes)
* Completion Percentage
* Device Type

### Dataset Summary

* 500 Customers
* 200 Content Titles
* 10.52K+ Watch Hours
* 5 Countries
* 8 Genres

# 🏗 Data Model

A Star Schema model was implemented using:

### Dimension Tables

* Dim_Customer
* Dim_Content
* Date Table

### Fact Table

* Fact_Viewing

Relationships were established using Customer ID, Content ID, and Date fields to optimize dashboard performance.
